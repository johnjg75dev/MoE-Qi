§moeqi-core

Core codec implementation and container formats.

Public entrypoints:

  • moeqi_core::format::binary::{encode, decode} for the MOEQI1 container
  • moeqi_core::format::json::{encode_bundle, decode_bundle} for debugging / inspection

§Build docs

From the repository root:

cargo doc --workspace --no-deps

§Example: encode + decode (binary)

use moeqi_core::{format, CodecConfig, Image, PixelFormat};

let img = Image {
    width: 2,
    height: 2,
    format: PixelFormat::Gray8,
    data: vec![0, 10, 20, 30],
};
assert!(img.validate());

let encoded = format::binary::encode(&img, CodecConfig::default())?;
let (decoded, _cfg) = format::binary::decode(&encoded)?;
assert_eq!(decoded, img);
