§moeqi

Convenience wrapper crate for the MoE-Qi codec implementation.

Most of the implementation lives in moeqi-core. This crate provides a small, stable surface:

  • moeqi::encode / moeqi::decode for the binary container format (MOEQI1)
  • moeqi::Image, moeqi::PixelFormat, moeqi::CodecConfig re-exports

§Build docs

From the repository root:

cargo doc --workspace --no-deps

§Quick start

use moeqi::{CodecConfig, Image, PixelFormat};

let img = Image {
    width: 2,
    height: 2,
    format: PixelFormat::Gray8,
    data: vec![0, 10, 20, 30],
};

let bytes = moeqi::encode(&img, CodecConfig::default())?;
let (roundtrip, _cfg) = moeqi::decode(&bytes)?;
assert_eq!(roundtrip.data, img.data);
